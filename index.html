About Me

My code so far has been focused on applying the ‘rules’ and structure of music to new environments and interfaces. I want to help people understand and manipulate music in a more accessible and physical way. I have explored areas such as stochastic systems, FFT effects, and motion capture. When I’m not coding or making music I’m tinkering and building in other ways. You can see some of my other projects here.

I am currently studying in a program called Thinkful where I am honing my skills in web-design and software development so I can help to make more user friendly and functional programs. CURRENT PROJECTS

RHYTHMODULUM

A web app for generating and manipulating rhythmic patterns. I built it as part of an ongoing set of tools for scripted music creation. My goal for this project is to make it easy for anyone to create highly specific or totally improvised enjoyable and intricate rhythmic data and percussion as quickly as possible. This was created with HTML, CSS, and JavaScript with original algorithms as well as implementation of the Web Audio API. Soon to come: MIDI Clock/Transport control, MIDI IN/OUT, and custom sample kits. Check it out or view the git below!

CHORALATOR

A web app that generates music with up to four voices by concatenating randomly generated motifs into larger musical forms. Currently available as an Ableton Live 'M4L' device.
